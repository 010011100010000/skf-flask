File name creation let you add new checklist and items, you even can reuse an item that is already been written.

file names logic of checklist items, for example:
31--owasp10--3.md

{id-of-file}--{checklist-name}--{kb-item}.md

id-of-file = position and number item
checklist-name = the name of your new checklist: a-zA-Z0-9
kb-item = refference to the issue Description and Solution
