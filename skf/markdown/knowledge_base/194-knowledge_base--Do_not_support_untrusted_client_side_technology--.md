Do not support untrusted client side technology
-------

**Description:**

when using unsupported client side technologies you have an high change of being compromised.
This is due the fact the code can be malicious and could do evil stuff to the user.


**Solution:**

Do not use Flash, Active-X, Silverlight, NACL, client-side Java or other client side technologies 
not supported natively via W3C browser standards.
